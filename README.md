# AWS Serverless Tutorial
Creating a serverless solution for auction website by using AWS.

## Resources
- API Gateway
- Lambda
- DynamoDB
- SQS
- S3
- CloudFormation (IAC)

## Getting started
```
sls create --name YOUR_PROJECT_NAME --template-url https://github.com/codingly-io/sls-base
cd YOUR_PROJECT_NAME
npm install
```
